# Oopa Genie

## Introduction

Oopa Genie is a all in one telegram bot. Without using any commands, it will work like chatGPT. You can talk to the bot [here](https://t.me/genie_oopa_bot).

## Features
- Horoscope
- Weather
- ChatGPT
- Reacts to messages
- Sends anime pics
- Sends anime quotes
- Recommends anime
